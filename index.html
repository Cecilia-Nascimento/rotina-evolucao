☁️ Conectar Google Drive

Cole seu Client ID do Google Cloud. Seus dados serão salvos automaticamente em rotina-evolucao-backup.json no seu Drive — sem precisar de servidor.

Como criar seu Client ID (gratuito, 5 min):
1. Acesse console.cloud.google.com e faça login
2. Clique em "Selecionar projeto" → "Novo projeto" → nomeie (ex: Rotina 2026) → Criar
3. No menu lateral: APIs e serviços → Biblioteca → busque "Google Drive API" → Ativar
4. Vá em APIs e serviços → Tela de permissão OAuth → Externo → preencha só o nome do app → Salvar
5. Vá em Credenciais → Criar credencial → ID do cliente OAuth 2.0
6. Tipo: Aplicativo da Web
7. Em "Origens JavaScript autorizadas" adicione: carregando...
8. Clique em Criar, copie o Client ID gerado e cole acima
